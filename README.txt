Scripts used to produce Mixed Migration Centre global updates on the impact of COVID-19 on refugees and migrants, see http://www.mixedmigration.org/resource-type/covid-19/

Numbers denote global update number. For example, 02_ denotes a file related to second global update.

For each update, _preparation denotes the file used to prepare the data, whereas _analyses denotes the file in which analyses were conducted. For example, the file 02_analyses contains the code used to analyse the data for the 2nd global update.

All figures are in the folders _update_files.

Rmd and md files contain the reports.

Actual data belongs to the MMC and is not shared here. Please also read acknowledgements at the end of each report.
